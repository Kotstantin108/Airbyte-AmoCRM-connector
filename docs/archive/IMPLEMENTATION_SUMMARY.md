### 🔴 **4 КРИТИЧЕСКИХ ПРОБЛЕМЫ**

| # | Проблема | Риск | Решение | Статус |
|---|----------|------|---------|--------|
| 1 | Несоответствие доменов (таблицы не созданы) | Синхронизация не работает | `CRITICAL_FIXES.sql` раздел 1 | ✅ Готово |

### 🟠 **3 ВЫСОКИЕ ПРИОРИТЕТЫ**

- Отсутствие индексов на `_synced_at` (производительность)
- Нет мониторинга ошибок (видимость)
- Нет проверки FDW перед началом (надёжность)

### ✅ **ЧТО УЖЕ ХОРОШО**

- Advisory locks (защита от параллельных запусков) ✓
- Dead Letter Queue для ошибок ✓
- Tombstone Shield для удалений ✓
- Проверка на большие батчи ✓
- Корректная логика UPSERT ✓
